Biodiversity in National Parks — Conservation Risk Analysis
Project Overview

This project analyzes biodiversity data from U.S. National Parks to identify patterns in species conservation status and ecological risk distribution. The objective was to determine which species categories face the greatest conservation pressure and which parks represent biodiversity hotspots requiring attention.

Objectives

Clean and prepare ecological datasets using Python

Merge species and observation records

Analyze conservation status distribution

Identify at-risk species categories

Compare protected species activity across parks

Communicate findings through data visualization and presentation

Dataset

The analysis combines two datasets:

Species Information — taxonomy and conservation status

Park Observations — recorded species sightings across parks

Tools & Technologies

Python (Pandas, NumPy)

Seaborn & Matplotlib

Data Cleaning & Feature Engineering

Exploratory Data Analysis (EDA)

Statistical Interpretation

PowerPoint Presentation Design

Key Insights

Conservation risk is unevenly distributed across species categories.

Certain parks function as biodiversity hotspots with concentrated protected species observations.

A small number of protected species account for a large share of ecological monitoring activity.

Data-driven prioritization can support conservation planning.

Project Outputs

Cleaned datasets

Merged analytical dataset

Python visualizations
