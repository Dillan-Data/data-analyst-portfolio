Olist E-Commerce Data Analysis
Project Overview

This project analyzes the Olist Brazilian E-commerce dataset to evaluate overall business performance, identify sales trends, and determine the most important revenue-driving product categories.

The project demonstrates an end-to-end analytics workflow, starting from raw relational data, performing data cleaning and aggregation using SQL, and building an interactive business intelligence dashboard in Tableau.

The final dashboard provides a clear overview of key performance indicators, monthly order trends, and product category performance.

Objectives

The main objectives of this project were to:

Clean and structure raw transactional data using SQL

Analyze overall revenue and order performance

Identify monthly order trends over time

Determine which product categories drive the most revenue and orders

Create a business-focused dashboard for decision-making

Tools & Technologies

SQL – data cleaning, joins, and aggregations

Tableau – data visualization and dashboard creation

CSV Dataset – processed dataset exported from SQL

Data Preparation

The original dataset consists of multiple relational tables including orders, customers, products, order items, and payments.

SQL was used to:

Join relevant tables into a unified dataset

Calculate total revenue per order

Aggregate metrics for dashboard analysis

Export the cleaned dataset as a CSV file for visualization

The processed dataset was then imported into Tableau to build the final dashboard.

Key Metrics

The dashboard summarizes the following business KPIs:

Total Revenue: $16,008,872

Total Orders: 99,441

Average Order Value: $154.10

These metrics provide a high-level view of overall e-commerce performance.

Key Insights

Analysis of the data revealed several important trends:

Order volume increased steadily throughout 2017, indicating strong platform growth.

Order activity remained relatively stable during 2018.

The Cama, Mesa e Banho category generates the highest number of orders and revenue.

Beleza & Saúde is the second-largest revenue contributor.

Some categories generate high order volume but lower revenue per order, suggesting differences in pricing or product mix.
