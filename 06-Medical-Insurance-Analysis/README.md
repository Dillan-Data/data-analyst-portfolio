# Medical Insurance Cost Analysis

## Author
Dillan Maart

## Objective
This project analyzes U.S. medical insurance costs data to uncover key patterns in charges based on factors such as age, BMI, smoking status, and region.

## Tools Used
- Python (csv module, custom functions)
- fpdf2 (for PDF export)

## Deliverables
1. Python script (`Insurance_Report_DillanMaart.py`)
2. Automated text report (`insurance_report.txt`)
3. Professional client-style PDF report (`Insurance_Report_DillanMaart.pdf`)

## Key Findings
- Smokers pay almost 4x more than non-smokers.
- Insurance costs increase steadily with age, especially after 50.
- The Southeast region shows the highest average insurance costs.
